Cachy
=====

Cachy provides a simple yet effective caching library.

.. toctree::
    :maxdepth: 2

* A simple but powerful API
* Thread-safety
* Decorator syntax
* Support for **memcached**, **redis**, **database**, **file**, **dict** stores

Cachy supports python versions **2.7+** and **3.2+**
